# Testing Guide

After you've completed a new feature and are ready to start testing, here are the steps you should take:

1. `git checkout <YOUR-BRANCH-NAME>`
2. `git checkout -b <YOUR-TEST-BRANCH-NAME>`
3. Work on your test code
4. `git add .`
5. `git commit -m "commit message"`
6. `git merge <YOUR-TEST-BRANCH-NAME>`
7. `git push -u origin <YOUR-TEST-BRANCH-NAME>`
